# Калькулятор рецептов

Сверху заголовок где написано что это калькулятор рецептов
Ниже название рецепта
Потом в ряд две панели форм: "Оригинальная форма" и "Целевая форма". Форма выбирается из выпадающего списка (круглая/прямоугольная), есть чекбокс "Открытая форма", и поля для размеров (для круглой: радиус и высота; для прямоугольной: ширина, глубина, высота).
Ниже в ряд три колонки: оригинальный рецепт, блок со стрелками и коэффициентами, пересчитанный рецепт. Коэффициенты отображаются с округлением до одного знака после запятой.
У рецептов есть 2 раздела: Тесто (Crust) и Начинка (Filling). Из левого crust в правый crust идёт стрелочка с коэффициентом пересчёта по площади. То же самое с filling, но там коэффициент пересчёта по объёму. Каждый ингредиент слева начинается с кнопки [×] для удаления, затем название, поле для ввода количества, единица. Под списком ингредиентов — форма добавления нового.
Под колонками с ингредиентами -- инструкция как делать
