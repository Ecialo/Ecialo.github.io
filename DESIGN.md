# Калькулятор рецептов

Сверху заголовок где написано что это калькулятор рецептов
Ниже название рецепта
Потом в две колонки рецепты. Слева оригинальный, справа с пересчитанными по коэфициентам ингридиентами.
Сверху рецепта указание формы для пирога. Она может быть круглая или прямоугольная. Размеры можно крутить.
У рецептов есть 2 раздела: crust и filling. Из левого crust в правый crust идёт стрелочка, на ней коэфициент по пересчета по площади. Тоже самое с filling, но там коэфициент пересчетп по обьёму.
Под вот этими двумя колонками с ингридиентами -- инструкция как делать
